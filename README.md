# Amazon Product Pricing & Demand Behavior Analysis

## Overview

This project analyzes Amazon product pricing using both cross-sectional and time-series approaches to understand pricing strategies and demand behavior.

---

## Project Workflow

1. Data Cleaning & Preprocessing (Python)
2. Exploratory Data Analysis (EDA)
3. Feature Engineering (Discount %, Demand Index)
4. Time-Series Modeling (Seasonality + Events)
5. Data Visualization (Tableau Dashboards)
6. Insight Generation

---

## Part 1: Market Snapshot Analysis

* Analyzed large-scale Amazon product dataset
* Identified category-level pricing, discount, and rating trends

### Key Insights

* Electronics and appliances dominate high-price segments
* Fashion categories show aggressive discounting
* Most products fall within mid-price range

---

## Part 2: Pricing Behavior Analysis

* Built a time-series dataset incorporating:

  * Seasonality
  * Promotional events (Diwali, Festive, Year-End)

### Key Insights

* Pricing is influenced by both seasonality and promotional events
* Event-driven discounts significantly increase demand
* Demand does not follow a simple linear relationship with price
